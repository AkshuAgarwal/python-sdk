###########
Data models
###########

This section explains data models used in topggpy to represent data received from Top.gg.

.. note::
    All listed models subclass :obj:`py:dict` and allow retriving data via attributes *and* keys (i.e., both ``response['id']`` and ``response.id`` are valid).
